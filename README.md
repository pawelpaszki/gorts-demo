# Bookshelf API

A demo Go web application for validating the gorts regression test selection tool.

## Quick Start

```bash
# Run the server
make run

# Run tests
make test

# Build binary
make build
```

## Purpose

This application is designed to demonstrate and validate regression test selection (RTS) effectiveness. The commit history is structured to:

1. **Commits 1-20**: Build the application incrementally with tests
2. **Commits 21-40**: Make targeted changes to validate RTS behavior
